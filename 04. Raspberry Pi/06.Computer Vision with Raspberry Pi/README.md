## 06. Computer Vision with Raspberry Pi

This chapter introduces **computer vision using Raspberry Pi**, with a focus on simple, practical, and beginner-friendly examples.

You will learn how to:
- Use **USB cameras** with Raspberry Pi
- Install and work with **OpenCV**
- Capture and process **images and video** in Python
- Build a strong foundation for **vision-based robotics projects**
